# contact-manager

## Setup
Das beiliegende "Setup.msi" kann verwendet werden um die Applikation zu installieren.


## Login
Für das login gibt es zwei vordefinierte Benutzer:
- Administrator: 
    - Benutzername: admin
    - Passwort: 1234
    - Rechte: Bearbeiten, Lesen
- Lese-Benutzer: 
    - Benutzername: readonly
    - Passwort: 1234
    - Rechte: Lesen
